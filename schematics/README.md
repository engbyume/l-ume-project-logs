# L-UME Schematics

This folder contains the electrical and mechatronic schematic designs for the final version of the L-UME (Light-based User Motion Engine) wearable device. 

## Schematic Files

- **L-UME Component Schematics (L-UME Componenet Schematics.png):** The detailed circuit schematic showing component connections between the Arduino Nano, the dual MPU6050 IMUs (with AD0 addressing for 0x68 and 0x69), the HM-10 BLE module (including the RX voltage divider), and the LED feedback array.
- **L-UME Raw Schematics (L-UME Raw Schematics.png):** The raw physical wiring and hookup diagram to reference during assembly and soldering of the final wearable brace.
