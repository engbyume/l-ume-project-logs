# CAD files and stuff

This section contains the 3D design files for the L-UME biometric system enclosures.

## Enclosure Descriptions

- **L-UME Main and Knee Units**: The primary housing for the wearable system. These units securely hold the prototype boards and IMU sensors, ensuring reliable biometric data collection while the user is in motion.
- **L-UME Battery Unit**: The power bank holster. This component holds the system's power bank in a stable position to provide consistent energy to the hub and sensor pods.

## Included Files

- **L-UME Main and Knee Units (STL)**: 3D printable mesh file.
- **L-UME Main and Knee Units (STEP)**: Standardized 3D model for editing in CAD software.
- **L-UME Battery Unit (STL)**: 3D printable mesh file for the holster.
- **L-UME Battery Unit (STEP)**: Editable 3D model for the holster.
