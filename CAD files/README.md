# CAD files

This directory contains the 3D design files for the L-UME biometric system enclosures.

## Enclosure Descriptions

- **L-UME Main and Knee Units**: These files represent the primary housing for the wearable system. They are designed to securely hold the prototype boards and IMU sensors in place, ensuring reliable biometric data collection during movement.
- **L-UME Battery Unit**: These files are for the power bank holster. This component holds the system's battery pack (power bank) in a stable position to provide consistent power to the hub and sensor pods.

Available in both STL (3D printable) and STEP (editable) formats.
