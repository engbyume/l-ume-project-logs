# Battery Unit

This folder contains the 3D design files for the L-UME power bank holster. This component holds the system's power bank in a stable position to provide consistent energy to the hub and sensor pods.

For the primary wearable housing that holds the prototype boards and IMU sensors, please see the [Main and Knee Units](../Main%20and%20Knee%20Units/).
